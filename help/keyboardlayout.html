Keyboard Layouts

Two types of nepali Keyboard Layouts are available till date: Traditional and Romanized Layout.

It is essential that unicode is supported in your OS.Check Unicode article .

Traditional layout

Traditional Layout is designed for True Type Font (TTF) like Preeti, Kantipur, etc users.

Romanized Layout

Romanized Layout is transliteration of the Nepali using English characters. The devnagari characters are mapped according
to the semi-phonetic transcriptionacross keyboards layouts. For example, the A,S,D,F,G,H keys corresponds to आ,श,ध,ऊ,घ,अ respectively (English sounds).
